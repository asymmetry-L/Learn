## Java概述



### 基本语法



```java

    public static void mian (String args[]) {
        //入口：main函数既是Java既是程序的起始点，也是结束点 
        System.out.println("hello world!");//输出
    }
```

### Java标识符

1.  必须以字母，下划线， $$开头，其他部分可以是数字字母的组合，但不能以数字开头

2.  标识符之间不能含有空格

3.  不能是Java的关键字【48个】

4.  标识符只能包含两个特殊字符，即美元符和下划线

5.  Java严格区分大小写

### 类的命名规范

1.  遵循Java本身的命名规范
2.  见名知意，常用取简，专用取繁
3.  如果是一个单词，单词首字母大写；如果是多个单词首字母大写
